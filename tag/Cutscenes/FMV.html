P4 Opening Movie (Patch)

P4G mod by Pixelguin
02/04/2021

This mod replaces the "Shadow World" opening movie with "Pursuing My True Self" from the original Persona 4. It works with both the High and Low quality cutscene setting.

Instead of replacing the movie files entirely with the mod loader, this is a simple Inaba patch that redirects p4ctop3_e.usm to p4ctop1.usm. This removes the need to include movie files in the mod at all and should improve compatibility for those having trouble with movie replacements.

Updated 10/18/2020
This mod has become outdated. A version of this mod that simply tells the game to play this video exists now, significantly reducing the file size.
You can download it here: https://gamebanana.com/gamefiles/14676

This is a mod for Persona 4 Golden that replaces Golden's current intro with the intro from the PS2 game.

How to Install:
IF YOU HAVE AEMULUS:
This mod is compatible with Aemulus.
Put the "PS2 Intro" folder in \Packages. Open Aemulus and enable the mod, refresh if it isn't showing up. After that build the mod, open the game through Reloaded, and now it should work.

If you don't use Aemulus:
Inside the "PS2 Intro" folder are folders called "movie00000" and "movie00002".
Place these two folders in \mods
Open the game through Reloaded and now it should work.
