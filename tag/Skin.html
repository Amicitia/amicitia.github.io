(WIP) Less Flat Stuff

P4G mod by Simoazzo
07/01/2021 (updated)

Updated 07/01/2021
The mod

Currently this mod has been developed by both me and Noodleslapper, with this mod,we want to replace lots of models from cutscenes and even field models, we've started from data00002/model/npc2 and selected around 130 files, but we might check them all again and add some to the list.

How does this work?
Working on this mod is really really easy just tedious and long, most of the models can be just ported from P5 and I've already painstakingly sorted all P5 models that can help out with this project.
To work on a model you just have to:
• Get the right model from P5
• Extract the model from P4
• Open it in the 3d editing software of your choice
• Align the HD one with the OG one
• Save it as .fbx and convert it with the model converter
• To test it you can use various methods, just get a save file, chose an easy to access cutscene and find an item from said cutscene, replacing that model will give you an idea of how the new model looks.

(WIP) CAS: Character Anime-Shading

P4G mod by Magusbear
03/23/2021

This is basically a texture conversion mod which aims to change all the character textures to ones resembling the anime/bustups. This is a work in progress and a massive and time consuming project, so don't expect to see it finished anytime soon.

This mod does not change the models or the shader, only the textures. This mod therefore will never look perfect but in my opinion it is still an upgrade over the original visuals. In a perfect world this would be coupled with a texture mod that changes the environment to the same style.
