Enable Debug Log by TGE

This cheat prints string output from the game's original debug functions into the PCSX2 log. TGE's modification also prints the filename and path of each file the game accesses in real time, which is extremely helpful for finding and narrowing down the location of assets like models, fields etc.
Copied cheat to clipboard!
// Enable debug log
patch=1,EE,00421DE8,word,00000000 // always branch even if not debug

Enable Debug Options in TV World

Disable Save File Integrity Check

This cheat allows you to load modified save files where the checksum doesn't match the contents. Useful for changing the player's name or stats.
Copied cheat to clipboard!
// Disable save file integrity check patch=1,EE,002A402C,word,00000000 // don't branch to crc check patch=1,EE,002A3FE8,word,10000004 // always branch to 0x002A3FFC regardless if v0 is zero
