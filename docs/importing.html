Importing

As seen in the Messagescript tutorial, in order to use text strings in a .flow, you must reference a .msg file to get the strings from.
An "import statement" usually looks like the following, at the top of the document: 
import( "field.msg" );

The paths are relative to the .flow file, so in this case, if your .flow is at C:\Users\Username\Documents\field.flow, it would look for field.msg in the Documents folder.

Once a .msg file has successfully been referenced, you can start referring to the names of messages in your .flow. When compiling, your resulting .bf will contain a .bmd made from your .msg.
