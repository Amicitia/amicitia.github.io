Static FPS Cap Changer

P5S tool by zettozoid
05/02/2021 (updated)

Updated 05/02/2021
Notice: Dynamic speed variant (no slowdown) available at https://gamebanana.com/tools/7059

Change the FPS cap without speeding up gameplay.

Press F1 to enable float speed (mandatory) then press the hotkey for the FPS.
F1 for 240fps
F2 for 120fps
F3 for 144fps
F4 for 75fps
F5 for 165fps
F6 for 60fps

Press hotkey after title screen to avoid softlock.

Known issues:
- Bustup animation is tied to FPS

Please reports any issues not found above yet, thanks!

FPS++|Dynamic FPS Uncap

P5S tool by zettozoid
05/02/2021 (updated)

Updated 05/02/2021
Uncap FPS (to 240) while avoiding slowdown caused by variable FPS

Only works on 1.02 Steam Version (latest)
If you only have v1.00, you can use the static fps one.

Hotkey:
F1 to enable (mandatory), set cap to 240
F2 to change cap to 120
F3 to change cap to 144
F4 to change cap to 75
F5 to change cap to 165
F6 to change cap to 60 (still using dynamic speed)

Possible issues (and workaround):
-Portrait animations are tied to fps
fix: None
-Unstable game speed
fix: Wait a few seconds or alt-tab twice
workaround: Use the static FPS cap changer

P3 Movie Music Portable Edition

P3P mod by Sate
04/27/2021 (updated)

Updated 04/27/2021
Replaces the game's music with songs from all four of the Persona 3 movies. Broadly speaking, they correspond to the same situations from the movies. (i.e. Persona Summoners plays while fighting Tartarus minibosses, since it played while fighting a Tartarus miniboss in the first movie)

This also has some artistic license just for fun, like Tartarus's themes being replaced with music from the movies based on whoever will be important during that block. Leftover tracks use the high quality versions from my other mod on here, to clear up any confusion. It also "adds" a few songs via messing with ambient sound loops (i.e. the Yakushima trip has music now, along with it's ocean and seagull sound effects)
