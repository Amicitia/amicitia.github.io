Classic Style Music Mod

P4G mod by Rob-RmanX
06/28/2020

Hey there! This is a mod to give the game a bit of retro-flare since Atlus made a special boss theme in a retro style but not the other songs. I even included a few other non-TV World songs since I was already putting this mod pack together. Either way you can enable or disable individual songs at your leisure.

Instructions are pretty simple. I do recommend the P4G Music Manager by T-Pose Ratkechi so if you'd like: download and set that up. The Music Manager is how I was able to put this all together so my instructions will follow suit.

INSTRUCTIONS:

1) Download the Music Manager

2) Download RmanX Music Mod Pack

3) Open/Setup music manager

4) Use the manager to replace the songs you'd like

5) Make a backup of your BGM.xwb file located in your P4G PC files

6) Copy the new BGM.xwb created by the manager

7) Place that new BGM.xwb in the P4G PC files where the original one was

8) Boot game

Pretty easy, right? This has been something I've personally wanted since the original P4/P ...

P4G mod by blivveries

Persona 4 Golden only loops the first verse of Time To Make History in-game. With the recent update to TGE's modloader, music mods are ridiculously simple, so I decided to replace it with the "Original Full Size Version" featured in P4D, which has the second verse and is currently the best "full" version of the theme that isn't also a remix.
