Skin 		The mod

Currently this mod has been developed by both me and Noodleslapper, with this mod,we want to replace lots of models from cutscenes and even field models, we've started from data00002/model/npc2 and selected around 130 files, but we might check them all again and add some to the list.

How does this work?
Working on this mod is really really easy just tedious and long, most of the models can be just ported from P5 and I've already painstakingly sorted all P5 models that can help out with this project.
To work on a model you just have to:
• Get the right model from P5
• Extract the model from P4
• Open it in the 3d editing software of your choice
• Align the HD one with the OG one
• Save it as .fbx and convert it with the model converter
• To test it you can use various methods, just get a save file, chose an easy to access cutscene and find an item from said cutscene, replacing that model will give you an idea of how the new model looks.

Download

Tags:
Misc 		I have decided to upload this tool I've been using for my Less flat Characters mod.

Here is a summary of what it does

• Path to the texture folder: this folder will have to contain the new textures you want to apply with the original name of the texture in .tm2 format (when replacing a model the material is the same as the original so you have to overwrite the original texture with yours), it's set to the texture folder in the tool when you launch it but it can be varied so that you can use multiple textures for different characters in different folders, putting too much stuff in this folder will only slow down the process but the results will always be the same because the model will only pick the textures that have the same name as the originals when compiling.

• Path to the Input folder: here you'll have to give the path to the original .amd files you want to edit, do not use files that you already edited with this process because it could result in some errors.
