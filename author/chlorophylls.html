Replacing Textures in P5S

P5S guide by chlorophylls
07/01/2021

Introduction
This guide is intended to show users how to replace textures/create texture mods for Persona 5 Strikers.

Let’s start with a basic texture replacement, like editing the color of Joker’s hair. I will be using this as the example throughout, but the same process will apply for any textures you want to edit.

For other textures, there are user-compiled articles listing textures (including UI textures) and their locations on the Amicitia wiki.

Tools Needed

• Cethleann - Essential for unpacking and decrypting. The Koei-Tecmo data tools. You can download the latest builds here at Appveyor.
• Gust Tools - More specifically, gust_g1t, for texture replacements (which are g1t files). Get the latest releases here.
• Noesis - For viewing the g1t textures. Needs the g1m/g1t/g1a importer plugin which you can find here.

• Aemulus - Not only does it manage P5S mods, it can also patch .rdb files the same way RDB Tool does.

After-Battle Ammo Refill

P5 mod by chlorophylls
06/02/2021 (updated)

Updated 06/02/2021
This mod makes it so that your ammo automatically replenishes after every battle, just like how it works in P5R.

Also adjusts the tutorial messages and cutscene to reflect the change.
For Mod Compendium
As A-NB has noted, this may conflict with other mods that replace dngPack.pac in field/, such as the Mod Menu. If that happens, use Amicitia to replace the support.bf file in that dngPack.pac and atDngPack.pac with the one in this mod (in the "BF Script" folder).

This mod is not compatible with the Afterbattle Replenish Mod.

Persona 5 Royal Bustups

P5 mod by chlorophylls
05/28/2021

This mod backports the new and redrawn portraits from P5 Royal to the PS3 version of P5, and adds the ones for new characters such as Kasumi, José, and Maruki, and more, for people to use in their own mods. This also adds the bustup portraits for the new winter outfits.

TBH, Rexis did like 90% of the work with the actual portrait backporting; the most I did was add the entries for the new ones and tweak and test the bustup params for placement for those, as well as backport the assist/navi bustups.

DO NOT BROWSE THE BUSTUP FOLDER IN BUSTUP PARAM EDITOR (OR WHATEVER PROGRAMS YOU USE TO VIEW BUSTUP PORTRAITS FROM .BINS) OR CALL PORTRAITS HAPHAZARDLY INGAME IF YOU DON'T WANT TO BE SPOILED FOR ROYAL.

This mod imports all the portraits from P5R, which means that there are spoilery bustup portraits in the folder.

For the 4K-compatible version of this mod, see this thread on Amicitia: https://shrinefox.com/forum/viewtopic.php?f=15&t=527

Costume Dialog Mod (WIP)

P5 mod by chlorophylls
8/25/2020


This mod restores the unused costume dialogue found in P5R, which has been translated from the originally-Japanese text to English, thanks to the efforts of various fan translators and editors.
The unused script originally also had no portraits except for the neutral one; these have been edited with portraits to better fit the dialogue. Keep in mind that this is a fan translation of unused dialogue.
Note that the only dialogue that has been fully translated, edited, and coded into this mod so far is for Haru, Yusuke, and Makoto, due to the differing speeds at which different translators and editors work. This mod is a beta/work in progress, and will be updated for the other characters when possible. (The next party member to be updated with translated dialogue is Ann.)
To use it, with an outfit equipped (on Joker and/or any of the party members), talk to any one of your party members at the dungeon entrance (not safe rooms) and select ""How's the clothes?"" (this option only pops up if Joker or the party member has a non-default costume equipped).
