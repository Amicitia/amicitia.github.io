Expanded DLC Outfits

P5 mod by DeathChaos, DniweTamp
3/22/2021
Backports Persona 5 Royal DLC Outfits as well as adding a few new custom outfits of my own, all with their own included DLC BGM Music too!
To obtain these new outfits, simply interact with the box in Joker's room in Leblanc just as you would for the regular DLC Outfits, note that even if you already have the regular DLC Outfits (or if you have no DLC at all), interacting with the box will still give you the new ones from the mod.
Update 3/22/21: Added the outfit set for Tracksuits that was removed from the game by developers
- Use Starlight Joker model with better weights (thanks to Saitsu)

GMD/GFS/GAP Import Maxscript

P5 tool by TGE, DeathChaos
12/14/2020
A maxscript for 3DSMax that allows you to import GMD models and GAP animations from Persona 5.
Update 12/14/2020:
v0.16:
- Added P5R key types (34 and 35, PSHalf and RSHalf).
- Added missing PRHalf keytype support (should make all Persona Dancing physics animations load now).
v0.17:
-Fixed Morph target mesh reading code, should now read all of them correctly.
-Added full support for loading Persona Dancing Morph Target animations
-Added Camera support for Persona Dancing, should load model and camera animation without issues
-Fixed weights being incorrectly applied in some instances;
-There should no longer be models with random unrigged vertices
-There should no longer be models with incorrect weights

Talk Animation Fix

PQ2 mod by DeathChaos, ShrineFox
6/7/2019
A bug in the English localization of PQ2 causes protagonists' mouths to no longer animate during voiced cutscenes.

This mod is meant to fix that bug. Note: this hasn't been heavily tested yet, so if you encounter any issues, let us know.
