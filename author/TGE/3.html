Enable Debug Log

P4 cheat by TGE
6/9/2018 (updated)

This cheat prints string output from the game's original debug functions into the PCSX2 log. TGE's modification also prints the filename and path of each file the game accesses in real time, which is extremely helpful for finding and narrowing down the location of assets like models, fields etc. https://i.imgur.com/5I5Vos8.png

P3FES Controllable Party Members

P3FES cheat by TGE
6/9/2018 (updated)

This cheat allows you to choose a skill or item on your party member's turn, similar to in Persona 4 and 5. Known side effects: the protagonist's fusion skills can be used by everyone on their turns.


EVTTool

P5 tool by TGE
5/28/2018

Drag an EVT file from Persona 5's event folder onto this EXE to convert it to an editable JSON. Edit with your favorite text processor (like notepad++) and drag it back onto the EXE to convert it back to EVT.

BFMessageScriptEditor

P3FES tool by TGE
1/1/2018

tool for extracting and replacing BMD files within BF scripts (as decompiled MSG files) without the need for recompiling the BF itself. Useful as an alternative to the compiler to avoid issues when all you want to do is simple text edits.

RMD Maxscript

P3FES tool by TGE
7/9/2017

A tool for loading model and animation data from RMDs in 3DSMax. Extract the included scripts, EXE and folder to the same directory together to extract textures when loading the model. Can be used with Amicitia for model hacking, but make sure to export with the axis set to Z-up instead of Y-up.
