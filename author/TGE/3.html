Debug Log

P3FES cheat by TGE
6/9/2018
This cheat prints string output from the game's original debug functions into the PCSX2 log. TGE's modification also prints the filename and path of each file the game accesses in real time, which is extremely helpful for finding and narrowing down the location of assets like models, fields etc.
// Enable debug log by TGE

patch=1,EE,00503768,word,00000000 // always branch even if not debug
patch=1,EE,005CDE83,byte,0A // patch file load print string
patch=1,EE,00100DAC,word,0C14896A // patch file load function call to debug print

EVTTool

P5 tool by TGE
5/28/2018
Drag an EVT file from Persona 5's event folder onto this EXE to convert it to an editable JSON. Edit with your favorite text processor (like notepad++) and drag it back onto the EXE to convert it back to EVT.
