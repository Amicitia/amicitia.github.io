The Strongest Animation Splitter

P4G tool by Pioziomgames
06/13/2021 (updated)

Updated 06/13/2021
A tool for splitting and converting animation data from AMD/GMO files.

To use it you just select the file type you wanna use at the top then drag a file or folder to the panel on the left then select the options you want on the right (if you don't select anything it will just give you the files in mds) and then you click the big button at the bottom and wait till it's finished and it will create a folder with your splitted animation.

Bad Apple!! mod

P4G mod by Pioziomgames
05/13/2021

IMPORTANT!!
This mod is not compatible with Aemulus
Why?
Well it replaces 26 643 files which makes aemulus building process super slow, but don't worry about the size of it cause it's less then 10 mb.
So to use it paste the provided data_e folder into your persona4golden/mods folder if you wanna use it with other mods paste the data_e folder after building with Aemulus.

If you don't know what bad apple is check out this video: https://www.youtube.com/watch?v=9lNZ_Rnr7Jc

So basically this is a black and white mod.

All characters and some important things(I needed to make something so you still see where are you going) are white and the rest is black.

If you have problems going somewhere remember that you can always press square to open fast travel menu.

Texture conversion fixes - made the textures always convert
