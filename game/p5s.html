Replacing Textures in P5S

P5S guide by chlorophylls
07/01/2021

Introduction
This guide is intended to show users how to replace textures/create texture mods for Persona 5 Strikers.

Let’s start with a basic texture replacement, like editing the color of Joker’s hair. I will be using this as the example throughout, but the same process will apply for any textures you want to edit.

For other textures, there are user-compiled articles listing textures (including UI textures) and their locations on the Amicitia wiki.

Tools Needed

• Cethleann - Essential for unpacking and decrypting. The Koei-Tecmo data tools. You can download the latest builds here at Appveyor.
• Gust Tools - More specifically, gust_g1t, for texture replacements (which are g1t files). Get the latest releases here.
• Noesis - For viewing the g1t textures. Needs the g1m/g1t/g1a importer plugin which you can find here.

• Aemulus - Not only does it manage P5S mods, it can also patch .rdb files the same way RDB Tool does.

More Colorful Battle UI

P5S mod by HaythamQuake
05/26/2021 (updated)

Updated 05/26/2021
If anyone remembers, my first mod for Persona 5 was Colorful Battle UI which was inspired by P5S and added some extra color to the battle sprites. Now here we are with Persona 5 Strikers and I thought to myself, "the game already has plenty of color, let's add MORE!"

Instead of using the colors I used for CBUI, I used the colors that are used per character, but just like CBUI, the sprites used when you're traversing the metaverse are also fully colored!

This mod also includes lipsum's fix to the kaleidoscope. If you're having trouble with it, please let me know.

Please enjoy~!

Static FPS Cap Changer

P5S tool by zettozoid
05/02/2021 (updated)

Updated 05/02/2021
Notice: Dynamic speed variant (no slowdown) available at https://gamebanana.com/tools/7059

Change the FPS cap without speeding up gameplay.

Press F1 to enable float speed (mandatory) then press the hotkey for the FPS.
F1 for 240fps
F2 for 120fps
F3 for 144fps
F4 for 75fps
F5 for 165fps
F6 for 60fps

Press hotkey after title screen to avoid softlock.

Known issues:
- Bustup animation is tied to FPS

Please reports any issues not found above yet, thanks!

FPS++|Dynamic FPS Uncap

P5S tool by zettozoid
05/02/2021 (updated)

Updated 05/02/2021
Uncap FPS (to 240) while avoiding slowdown caused by variable FPS

Only works on 1.02 Steam Version (latest)
If you only have v1.00, you can use the static fps one.

Hotkey:
F1 to enable (mandatory), set cap to 240
F2 to change cap to 120
F3 to change cap to 144
F4 to change cap to 75
F5 to change cap to 165
F6 to change cap to 60 (still using dynamic speed)

Possible issues (and workaround):
-Portrait animations are tied to fps
fix: None
-Unstable game speed
fix: Wait a few seconds or alt-tab twice
workaround: Use the static FPS cap changer
