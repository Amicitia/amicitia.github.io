Misc 		Introduction
This guide is intended to show users how to replace textures/create texture mods for Persona 5 Strikers.

Let’s start with a basic texture replacement, like editing the color of Joker’s hair. I will be using this as the example throughout, but the same process will apply for any textures you want to edit.

For other textures, there are user-compiled articles listing textures (including UI textures) and their locations on the Amicitia wiki.

Tools Needed

• Cethleann - Essential for unpacking and decrypting. The Koei-Tecmo data tools. You can download the latest builds here at Appveyor.
• Gust Tools - More specifically, gust_g1t, for texture replacements (which are g1t files). Get the latest releases here.
• Noesis - For viewing the g1t textures. Needs the g1m/g1t/g1a importer plugin which you can find here.

• Aemulus - Not only does it manage P5S mods, it can also patch .rdb files the same way RDB Tool does.
