- Hassou Tobi

- Ragnarok

- Thunder Reign
- Panta Rhei
- Samsara

Light/Dark Neutral mod

P4G mod by JayTheTaco
06/19/2021 (updated)

Updated 06/19/2021
Have a mod that adds damage-dealing light/dark skills, but then you found out that they don't work on enemies that block dark/light, such as bosses? Ever wanted to just cheese a boss with an insta-kill move? Well, this mod is for you!

The Light/Dark Neutral mod is a simple mod that makes most bosses neutral/resistant to Dark and Light skills. There are some exceptions like Shadow Rise and Kasumi-no-Okami. The reasons being the fact that one battle is scripted, and I didn't know if insta-killing a scripted boss would cause problems. and for the latter, it's because the boss' entire gimmick is reflecting every element.

This is my first real mod so bare with me if there are some bugs.

How to Use
Prerequisites

• For preappfile to unpack for Persona 4 Golden, you'll need .NET Core 3.1 Desktop Runtime x64.
• For YACPKTool to unpack/pack cpk's for Persona 5, you'll need Microsoft Visual C++ 2010 Redistributable Package (x86)
• For Persona 4 Golden on PC, make sure you've set up Reloaded-II and the P4G mod loader first and foremost.
• For Persona 3 FES, setup HostFS for mod loading.
• For Persona 5, setup the proper patches for mod loading from mod.cpk. Just ignore the Setting Up Mod Compendium section since you'll be using Aemulus instead.

The Strongest Animation Splitter

P4G tool by Pioziomgames
06/13/2021 (updated)

Updated 06/13/2021
A tool for splitting and converting animation data from AMD/GMO files.

To use it you just select the file type you wanna use at the top then drag a file or folder to the panel on the left then select the options you want on the right (if you don't select anything it will just give you the files in mds) and then you click the big button at the bottom and wait till it's finished and it will create a folder with your splitted animation.
