Talk Animation Fix
PQ2 Mod by DeathChaos, ShrineFox
6/7/2019
A bug in the English localization of PQ2 causes protagonists' mouths to no longer animate during voiced cutscenes.

This mod is meant to fix that bug. Note: this hasn't been heavily tested yet, so if you encounter any issues, let us know.

PQ2 MC Name Patch
PQ2 Mod by DeathChaos
5/26/2019
A set of assembly patches (patched in thanks to Magikoopa) for Persona Q2 (USA) that patches out the player name functions to allow the longer "canon" names that normally the player cannot fit into the name input box such as Yu Narukami to display via hardcoded name calls the same way other characters do.
