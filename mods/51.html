Slightly Enhanced FMV Mod

P4G mod by pelvisbass
6/17/2020


The goal of this mod is to make the FMV videos look slightly better however there is only so much that can be done in its current state. For some reason I will never understand atlus decided to encode the FMVs in game with the WMV2 Codec rather than the much more modern AVC codec used for the psvita version of the game. This results in some really bad compression and blocking which in some cases can be distracting. So I decided to use vapoursynth to try my best to fix it up. The results are not perfect as there is a small amount of detail loss and blockiness however in my opinion the results are much better than the originals. If you are interested in it I left the script I used for vapoursytnh as well as my ffmpeg settings in the readme.

- Most Cutscene models

- Most main cast models

- Every persona model

- Every NPC model (including... adachi)

HOW TO INSTALL: 

(Before doing anything, extract Adachi.rar)

1. Before extracting make a new folder named data00002 then inside that folder make one named model

2. Now extract npc2.rar, and place the np2 folder inside the model folder. Repeat with npc.rar, and persona.rar.

3. rename your data00002.pac in your game files to data0002old.pac

4. Now extract tools.zip, and open up pac editor.exe(edited)

5. follow those steps, in summary 

Drag data00002old.pac onto the pac editor, it will freeze be patient

go into the persona, npc, or npc 2 folders, hit ctrl + a then drag those onto the pac editor, it will freeze each time you drag the files from those folders onto the pac editor
