Controller UI Overhaul

P4G mod by Pixelguin
05/28/2021 (updated)

Updated 05/28/2021
This mod replaces the controller UI with all-new icons that are higher-quality, more legible, and colorful!

The default button prompts in P4G are very lazily done: the fonts are inconsistent or hard to read, the shapes of the dpad and triggers can be incorrect, and all of them lack color. This mod aims to change that with a completely custom controller UI.

Compatible with all controller types (Steam Controller users will need to configure their gamepad in Big Picture settings).

Note: If you are using a next-generation controller (Xbox Series or PS5) and the game is not responding to button presses, open the Steam Properties window for P4G and set Steam Input Per-Game Setting to Forced On.

• Bigger, more legible font used for all text
• Classic white "Check!!" UI (P4G used a red accent, but this only makes sense on the Japanese Vita version where it represented the circle button)

P5 Strikers Font

P4G mod by Pixelguin
05/28/2021 (updated)

Updated 05/28/2021
Persona 4 Golden has a wider font than the PS2 version of Persona 4, presumably to make the text easier to read on the small PS Vita screen. However, it's much too large for a PC monitor or even a television screen.

This mod replaces P4G's font with Linux Biolinum Bold, the font from Persona 5 Strikers. Each character was individually typed and spaced using Photoshop and PersonaEditor, which also fixes some overlapping glyph issues from the original font. Some characters have also been adjusted to fit texture height limitations.

Resolution Fix

SMT3 mod by bertie117
05/25/2021

For some reason Atlus capped the resolution at 1080p for this game. The resolution in the options only affects the HUD and UI, while the render scale option in the settings affects the game itself, and there are only three options: Low, which is 360p, medium, which is 720p, and high, which is 1080p. Considering this is a PS2 game capped at 30fps, you probably have a lot of GPU power going to waste.

I made this mod with the help of The Rhythm Thief on the steam forums:

https://steamcommunity.com/sharedfiles/filedetails/?id=2494134877

To install, download the file for whatever resolution you want and replace the "GameAssembly.dll" file in the game's directory with it. The resolution will replace the "High" preset; the others are untouched. Make sure to backup the original GameAssembly.dll file.
