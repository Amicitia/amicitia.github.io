Persona 5 Royal Red Morgana Bus Backport


12/16/2020



This turns the Morgana Bus red, so it might affect cutscenes. The mod is setup so that the royal decal is seperate from the bus itself. So you can use the mod to have a red Morgana Bus without the decal. To enable the decal, go to the entrance of Mementos and select the ""celebratory"" decal. If you do not want the red bus, delete the c0100_051_01.GMD file in data\model\character\0100\.
