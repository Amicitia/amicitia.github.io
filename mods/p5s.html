More Colorful Battle UI

P5S mod by HaythamQuake
05/26/2021 (updated)

Updated 05/26/2021
If anyone remembers, my first mod for Persona 5 was Colorful Battle UI which was inspired by P5S and added some extra color to the battle sprites. Now here we are with Persona 5 Strikers and I thought to myself, "the game already has plenty of color, let's add MORE!"

Instead of using the colors I used for CBUI, I used the colors that are used per character, but just like CBUI, the sprites used when you're traversing the metaverse are also fully colored!

This mod also includes lipsum's fix to the kaleidoscope. If you're having trouble with it, please let me know.

Please enjoy~!
