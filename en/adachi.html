The Completely Self-Centered Cabbage Detective returns!

This time he stars in his own quest, investigating society's most twisted hearts.
Experience the highly acclaimed RPG again from a new angle. Follow the instructions below to get started!

Features
  • New player model
  • Start the game with Magatsu Izanagi
  • New art for the user interface
  • New voice clips
  • New BGM
  • Several different costumes to try out

Support Atlus by purchasing Persona 5 on Playstation Network or Amazon.

Installation Instructions

There are two different setup methods.

Custom Firmware on Playstation 3

  1. Obtain a compatible PS3 console ver 3.55 or lower and follow these instructions
  2. Use a homebrew application that enables FTP (File Transfer Protocol) on your PS3
    examples: CCAPI or webMAN MOD
  3. Take note of your console's IP in Network Settings and connect on PC
    with an FTP program such as Filezilla
  4. Navigate to dev_hdd0/game/NPJB00769/USDIR and move your downloaded mod.cpk file inside
  5. Patch the EBOOT.BIN at that location to support mod loading! More info here.
  6. Start the game and enjoy!

RPCS3 (Playstation 3 Emulator)

  1. Download RPCS3
  2. Install a PS3 firmware
  3. Install Persona 5 (and the DLC, if you have it).
  4. Put your RAP file for Persona 5 in RPCS3\dev_hdd0\home\00000001\exdata
  5. Navigate to dev_hdd0/game/NPJB00769/USDIR and move your downloaded mod.cpk file inside
  6. Replace the patch.yml file in the RPCS3 root folder with this one.
  7. Start the game and enjoy!
