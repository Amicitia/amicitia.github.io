Using PNACH Files
Once you have PCSX2 set up, you can use .pnach files to load cheats. In many cases, "cheats" can be very practical for modding purposes, as they are essentially just patches that change data in the game's memory while it's running. The effects of cheats are non-permanent and harmless (assuming you don't save after modifying things like EXP or time travel, which would of course persist after the cheat is disabled.)

To create a new cheats file for PCSX2, just navigate to your Documents\PCSX2\cheats\ folder and create a new .txt document.
Rename the file extension from .txt to .pnach and open it with a text editor, like notepad.
Get your cheats from wherever you'd like. there are several pages of them available on the forum.
A .pnach format cheat looks like the one on the left. You can copy/paste them into your file and save it.
You can label each one by putting two slashes before the name on a separate line. This prevents the emulator from reading that line.
In case you want to disable a cheat (but keep it in the file for later), you can put slashes in front of each line of the cheat as well.
In the System tab in the top left corner of PCSX2, ensure that you have Enable Cheats checked. To see if your cheats were found and applied, go to Misc > Show Console before booting the game.
While you're there, check the game's CRC at the top of the window. You will need to do this to know what to name your pnach file.
A CRC is like a hash of the game's ELF file: it changes whenever the executable is modified, so depending on your region or any mods you have installed, it will be different and your pnach file will have to be renamed.
