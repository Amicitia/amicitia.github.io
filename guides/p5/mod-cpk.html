Loading Mods in Persona 5

What's a Mod.cpk?

A mod.cpk file is a prepackaged archive that the game can load mods from. The files inside override the contents loaded from the game's other CPK files.
In order for this to work, you need to either replace the game's EBOOT or patch it with this YML file made by TGE.


Download the EBOOT for your region and CFW type. If there isn't one for your region, see this guide on patching the EBOOT yourself.
Or, if you're using an emulator,
download the patch.yml for RPCS3 to get started.

Using mods on a Custom Firmware PS3

Presently, the mod.cpk patch does not seem to be compatible with Blu-ray disc versions of the game on CFW. Only the digital PSN release can take advantage of mod.cpk support on console. In order to install it on a real PS3, you need a custom firmware and an FTP homebrew like webman or multiman
as well as a client like Filezilla on PC.

Note your PS3's IP address (you can change this in Network Settings) and use it to connect to your FTP client.
Go to dev_hdd0/game/GAMEID/USRDIR and place the EBOOT file there, overwriting the original. The Game ID depends on your region, but should be one of the following:
NPEB02436 (Europe) / NPUB31848 (US)

If the game ID starts with "BL" instead (for instance, BLUS31604) then you need to place modded files in dev_hdd0/disc/GAMEID/PS3_GAME/USRDIR.

How to use the Mod.cpk

Simply drop the downloaded CPK into the USRDIR folder (same place where you replaced the Eboot) and you're good to go! Note that you will need to restart the game to see any changes, and that you cannot rename the CPK.

Using mods on RPCS3
In order to test mods on PC, you need RPCS3 downloaded and running on your PC. You will also need to install the game's PKGs and install a PS3 firmware. Rebug CFW works fine for this purpose.
Place your downloaded patch.yml file in the root of the RPCS3 directory. Open it with Notepad or Notepad++ and scroll to the bottom. Here, you'll need to replace the PPU Hash with your unique hash generated by RPCS3. To get the hash, right click the console and set it to "Trace." Then, start the game and quickly close it. Select all and copy the text in the console to notepad and search for "hash." This is what you'll be using.

Enabling/Disabling Eboot mods

At the bottom of the YML file, you can put a # symbol in front of the the same lines where you replaced your PPU Hash to disable those specific mods next time you start the game.

How to use the Mod.cpk

Simply drop the downloaded CPK into the game's USRDIR folder and you're good to go! Note that you will need to restart the game to see any changes, and that you cannot rename the CPK.
You can also patch EBOOT files using patch.yml files if you want to use RPCS3-specific mods (like the randomized music mod) on a CFW PS3.
