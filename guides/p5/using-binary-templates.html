About TBL Files
*.TBL Files are sets of tables that contain statistics used in Persona games. In Persona 5, they can be found in data.cpk/battle/table.pac.

Each TBL's set of data is related to its function. For example, ENCOUNT.TBL has variables for each set of enemies in the game-- their IDs, the music, the battlefield etc.

Setting Up the Binary Template
Right now, a work in progress binary template is available that can edit the PERSONA.TBL from Persona 5, as well as potentially some other ones.

Download it here.
Binary Templates are for use specifically with 010 Editor, an advanced hex editor with scripting features that really come in handy for formats like this.
(Note: 010 Editor has a 30 Day trial. We don't host warez here so you'll have to look elsewhere.)
Once you have 010 Editor installed and the Binary Template downloaded, open 010 Editor and navigate to Templates > View Installed Templates...
Then, click Add and select the P5_TBL.bt file.

Editing a TBL with the Template
Open your TBL file in 010 Editor (after extracting it from table.pac using Amicitia).
Then, navigate to Templates > Run Template. Choose P5_TBL. The hex digits should become highlighted with a color, and a table should expand at the bottom of the window.
Click the arrows next to the name of a structure to expand a list of table entries within it. There may be a lot of yet-unknown values, but important ones should already be labelled. For instance, here we see the values for the selected Persona's Arcana, Level, Strength and other stats.

By looking on the Wiki, we can determine which Persona corresponds to what number, as well as Arcanas and Skills.

When you're done editing these stats, press Ctrl + S to save the changes to the file. You can then replace the TBL in table.pac and add it to your mod.cpk using the Mod Compendium.
