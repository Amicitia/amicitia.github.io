About SD2VITA

What's SD2VITA?

SD2VITA is an unofficial MicroSD Card adapter that you can use with your exploited PSVita (as long as it's running a firmware on or between 3.60 and 3.68). For a guide on exploiting your Vita, see the following:

This guide if you're on 3.60 or 3.65
or this guide if you're on 3.67 or 3.68

How does it work?

By running gamesd.skprx, skprx module (or "plugin") through Henkaku, you can mount your SD card as a replacement for the system's memory card partition (ux0). The only drawback is that it uses the game card slot (gro0), so you won't be able to use physical game cartridges while the SD card is mounted. On the other hand, if you already have an official Sony memory card for the Vita, you can use it alongside the adapter for even more storage. Keep in mind that whichever one is mounted as ux0 is where games have to be installed, though.

Setting up SD2VITA
Download gamesd.skprx from this Github repository's releases page.

With henkaku enabled, run Vitashell and start an FTP server. Navigate to your ux0:tai folder. It should have a config.txt file. You may either transfer it to your computer, or use Vitashell's built-in text editor to replace all instances of ux0 with ur0. Also be sure to add the following line under *KERNEL in the config.txt:

ur0:tai/gamesd.skprx

Once the config.txt is edited and/or replaced to include those changes, transfer the tai folder and everything in it from ux0 to ur0.
Next, while still connected to FTP, back up everything in your ux0 directory to your computer. You should be able to do this by simply dragging it out of the FTP window and onto your desktop. Once you have a full backup, you may copy that to your microSD card. At this point, you may now shut off your PSVita, insert the SD card (with the adapter), and turn it back on to see the changes.
