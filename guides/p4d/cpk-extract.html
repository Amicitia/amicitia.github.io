Extracting P3P's CPK Files

What's a CPK?

A CPK file is an archive made by Criware that contains the game's content. They can be obtained and extracted several ways, as well as repacked.

In order to modify or swap the game's files, you must unpack the files contained in the CPK to a folder on your computer. Then, after your changes have been made, compress the extracted folder back into a CPK file.

Getting the CPK files from the game
Persona 4 Dancing has three main CPKs, data.cpk (containing most of the game data) and vita.cpk (containing sound and video), as well as patch.cpk which comes with an update that enables DLC. Using a Vita running Henkaku on 3.60 (see here for more information), you can extract the CPK from a cartridge or digital version of the game using Vitashell.
Simply open Vitashell, navigate to the ux0:/app/GAMEID folder and mount the directory. You can then copy it elsewhere and use FTP to transfer it to your PC.

Extracting the CPK contents
Next, you must follow these steps to get the files from data.cpk.

Download and extract CriPakGUI.zip. Run CriPakGUI.exe and click the Extract me! button to browse for your CPK file. It will prompt you to choose a location for the extracted contents. Then sit back and wait for it to finish!

Packing a CPK

If you want to mod the game instead of just browsing its files, follow this guide on how to use TGE's Mod Compendium program to generate new cpk files.
