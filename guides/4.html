Resolution Fix

SMT3 guide by bertie117
05/25/2021

For some reason Atlus capped the resolution at 1080p for this game. The resolution in the options only affects the HUD and UI, while the render scale option in the settings affects the game itself, and there are only three options: Low, which is 360p, medium, which is 720p, and high, which is 1080p. Considering this is a PS2 game capped at 30fps, you probably have a lot of GPU power going to waste.

I made this mod with the help of The Rhythm Thief on the steam forums:

https://steamcommunity.com/sharedfiles/filedetails/?id=2494134877

To install, download the file for whatever resolution you want and replace the "GameAssembly.dll" file in the game's directory with it. The resolution will replace the "High" preset; the others are untouched. Make sure to backup the original GameAssembly.dll file.

Save Everyday

P4G guide by AnimatedSwine37
05/23/2021

A mod that makes the save prompt to show up at the beginning of every day (just before early morning starts). 
(Requested by Emphabet on discord)

Compatibility (Please Read!)
This mod has now been added to Custom Sub Menu so if you are already using that you can just enable it in its menu. Also, if you are using Remove Lunchtime Invitations or Remove Hot Springs & Camping Trip Events please use Custom Sub Menu (which contains all of the mods as options) as those mods are incompatible with this.

A Known Issue
When saving with this the location you are in will not be labeled correctly in the save. This has no effect on gameplay, it is just a visual glitch. I may be able to fix it but for now, you'll have to live with it.
