RMD Model & Animation Import Maxscripts

 A tool for loading model and animation
data from RMDs in 3DSMax.
Extract the included scripts, EXE and folder to the same
directory together to extract textures when loading
the model.
Can be used with Amicitia for model hacking, but
make sure to export with the
axis set to Z-up instead of Y-up.
